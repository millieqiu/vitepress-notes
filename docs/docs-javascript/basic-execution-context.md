# [basic] Execution context

> 🌟 重要觀念！！

JavaScript 需要跑在引擎（engine）上才能運行。每一個瀏覽器都有自己的 JavaScript 引擎，其中以 Chrome 的 V8 最為知名，其目的是將你的 JavaScript 轉換為電腦懂的機器碼，來運行你的程式。

當程式碼準備開始運行時，會先建立兩種執行環境：

1. 全域執行環境
2. 函式執行環境
